# 🚀 이지케어(EasyCare) SaaS 플랫폼 개발 로드맵

> **"복잡한 케어를 간편하게, 데이터로 증명하는 가치"**
> 이지케어는 실버 케어 및 아동 발달 센터를 위한 지능형 통합 관리 솔루션으로 진화합니다.

## 💎 이지케어 필수 탑재 프로그램 (Core Modules)

이지케어 시스템이 "통합 솔루션"으로 기능하기 위해 반드시 포함되어야 하는 5대 핵심 모듈입니다.

```mermaid
mindmap
  root((EasyCare SaaS))
    1. 운영 관리 (Admin)
      일정/출석 관리
      수납/결제 시스템
      직원/급여 관리
    2. 스마트 케어 (Therapist)
      상담 일지 작성
      개별화 교육계획 (IEP)
      발달 평가 리포트
    3. 소통 포털 (Parent)
      알림장/공지사항
      가정 연계 활동
      성장 그래프 확인
    4. 성장 지원 (Growth)
      AI 마케팅/블로그
      잠재 고객(Lead) 관리
      지점별 통계 분석
    5. 마스터 콘솔 (Super Admin)
      멀티 지점 중앙 제어
      통합 매출 대시보드
      자동 배포 시스템
```

### 1️⃣ 운영 관리 프로그램 (Center Admin)
> **"센터의 행정 업무를 90% 줄여주는 자동화 비서"**

| 기능 | 설명 | 필수 여부 |
| :--- | :--- | :---: |
| **스마트 스케줄링** | 치료사/아동/치료실 충돌 방지 및 드래그 앤 드롭 일정 관리 | ⭐⭐⭐ |
| **자동 수납/청구** | 바우처, 카드, 현금 결제 관리 및 미납 알림 자동 발송 | ⭐⭐⭐ |
| **직원 근태/급여** | 수업 시수에 따른 급여 자동 계산 및 명세서 발송 | ⭐⭐ |
| **전자 문서 관리** | 입소 신청서, 개인정보 동의서 등 종이 없는 전자 서명 | ⭐⭐ |

### 2️⃣ 스마트 케어 프로그램 (Expert Tool)
> **"선생님은 아이에게만 집중하세요, 기록은 AI가 돕습니다."**

| 기능 | 설명 | 필수 여부 |
| :--- | :--- | :---: |
| **전자 상담 일지** | 음성 인식 및 키워드 선택을 통한 빠른 일지 작성 | ⭐⭐⭐ |
| **발달 평가 도구** | K-CDI, DEP 등 표준화된 검사 도구 디지털화 및 결과 시각화 | ⭐⭐⭐ |
| **IEP (개별화 교육)** | 아동별 장단기 목표 수립 및 달성도 추적 그래프 | ⭐⭐ |

### 3️⃣ 학부모 소통 앱 (Parent Portal)
> **"우리 아이의 성장을 실시간으로 확인하는 투명한 창"**

| 기능 | 설명 | 필수 여부 |
| :--- | :--- | :---: |
| **모바일 알림장** | 그날의 수업 사진, 영상, 핵심 피드백 실시간 푸시 알림 | ⭐⭐⭐ |
| **가정 연계 활동** | 센터에서 배운 내용을 집에서도 할 수 있도록 미션 제공 | ⭐ |
| **성장 리포트** | 매월 아이의 발달 변화를 분석한 AI 리포트 자동 전송 | ⭐⭐ |

### 4️⃣ 마스터 & 프랜차이즈 관리 (Super Admin)
> **"하나의 본점에서 전 세계 지점을 클릭으로 관리"**

## 🛠️ 단계별 상세 구현 계획

### 🟢 Phase 1: 기반 강화 (현재 완료 단계)
**"SaaS로서의 무결점 엔진 완성"**
*   **[완료]** 지점별 데이터 완벽 격리 (Multi-tenant RLS)
*   **[완료]** 실시간 브랜드 컬러 및 로고 변경 시스템
*   **[완료]** 슈퍼 어드민/지점 어드민/치료사/부모 권한 분리

### 🟡 Phase 2: 프랜차이즈 가속화 (2026년 2월~3월)
**"1,000개 지점도 클릭 한 번으로 관리"**
*   **지점간 데이터 이동(Portal)**: 슈퍼 어드민이 여러 지점을 한 눈에 모니터링하고 즉시 이동하는 기능.
*   **커스텀 도메인(CNAME)**: `gangnam.easycare.com` 처럼 각 지점만의 도메인을 자동 연결.
*   **센터별 설정 템플릿**: 신규 지점 오픈 시 프로그램, 설정값이 자동으로 세팅되는 '스피드 셋업' 기능.

### 🔴 Phase 3: AI 지능형 케어 (2026년 4월~6월)
**"일은 줄이고, 퀄리티는 높이고"**
*   **AI Smart Note**: 치료사가 키워드만 입력하면 서술형 상담 일지를 자동 완성.
*   **예측 분석 대시보드**: 수업 이탈 확률이 높은 아동/노인을 미리 감지하여 알림.
*   **AI 마케팅 비서**: 주말마다 센터 홍보 블로그 글과 인스타그램 문구를 AI가 알아서 작성 및 예약 발행.

### 🟣 Phase 4: 에코시스템 통합 (2026년 7월~)
**"케어의 시작과 끝을 연결"**
*   **Care-Pay**: 학부모/보호자가 앱에서 즉시 결제하고, 영수증을 자동 카톡 발송.
*   **스마트 알림장**: 아이의 오늘 하루 활동 사진을 AI가 예쁘게 편집하여 알림장으로 송부.
*   **Hiring Portal**: 센터 내 부족한 인력을 즉시 매칭해주는 치료사 매칭 시스템.
